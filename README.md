### 背景
+ 当前代码是阿里云企业工作台为第三方合作伙伴提供的，基于深度集成的合作模式的三方库
+ 本三方库提供了对三方插件在售卖完整生命周期（包含新购、变配、续费、退款、释放，服务启停）的回调函数
